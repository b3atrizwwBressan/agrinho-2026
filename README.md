# agrinho-2026
Projeto agrinho: agro forte e futuro sustentável 
 Esse trabalho foi feito com o objetivo de fazer com que o agricultor/usúario seja avaliado, de modo que, possa no final aprender oque não sabia. Foi pensado inicialmente para ser um jogo em pixel, tendo como inspiração: Stardew Valley, Celeste e undertale. 

 Foram utilizados: Piskel, Libresprite, Ibis paint, canva (para imagens); Gemini (para certas consultas de códigos)
